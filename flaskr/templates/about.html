Wilhelm and Alexander von Humboldt

The goal of this tool, Humboldt, is to enable linguists to explore language variation at a large scale. It is named after the Humboldt brothers, because like them, it combines linguistic knowledge and scientific exploration.

Humboldt gives you a web interface that allows the user to query for lexical phenomena in several languages, and to get both statistical analysis and map representations of the results along several demographic factors.

The interface gives linguists capabilities that would otherwise require programming experience, training in statistical methods, knowledge of geographical information systems, and expertise in natural language processing. We provide an interface to explore large-scale language variation along multiple socio-demographic factors - without any programming knowledge. It uses of large amounts of data and provides statistical analyses, maps, and interactive features that will enable scholars to explore language variation in a data-driven way.

Where does the data come from?

Currently, the results are based on web data from a review site. The nature of the data we use naturally has an effect on the searches it supports: some words are not used in reviews. However, we plan to add more sources as time goes on, to cover a wide range of domains and topics.

Example case: definitely

There are more than 300 variants of "definitely" in British English, including definately, deffinitely, or defanately. One of the more common variants is definately. One might wonder whether this is simply a random mistake, or whether there are any underlying variable governing it. A typing error seems unlikely, given that the differing keys are separated on the keyboard.

Looking for differences in age and gender (see below) does not show any substantial differences in use.


Figure 1: Frequency of definitely/definately spelling variations in English data by age and gender

However, when plotting the regional distribution on a map, we can see that the occurrences are centered around the regions of the Midlands (see Figure below). This makes a dialect-phonological feature a likely source for the variant.
